Investigación sobre CSS

1. Definición de CSS

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje de diseño utilizado para describir la presentación de documentos HTML. Permite separar la estructura del contenido y la presentación visual, facilitando así el diseño y la maquetación de páginas web de forma coherente y eficiente.

2. Funcionamiento Básico de CSS

CSS funciona a través de un conjunto de reglas que se aplican a los elementos HTML. Cada regla contiene un selector que identifica a qué elementos se aplica y un bloque de declaración que especifica las propiedades y los valores a modificar. Esto permite a los desarrolladores estilizar el contenido sin afectar su estructura semántica.

3. Incluir CSS en un archivo HTML

Hay varias formas de incluir CSS en un documento HTML:

  • Inline: Se aplica directamente a un elemento usando el atributo style. Ejemplo: <h1 style="color: red;">Título</h1>.
  • Interno: Se define dentro de una etiqueta <style> en la sección <head> del documento. Ejemplo:
    • <style>
    h1 { color: blue; }
    </style>
  • Externo: Se utiliza la etiqueta <link> para enlazar a un archivo CSS externo, lo que es la práctica recomendada. Ejemplo: <link rel="stylesheet" href="styles.css">.

4. Definición de una regla CSS

Una regla CSS es una declaración que define cómo se deben estilizar ciertos elementos HTML. Consiste en un selector que apunta a uno o más elementos y un bloque de declaración que incluye uno o más pares de propiedad-valor. Por ejemplo:

p { color: green; font-size: 16px; }

En este ejemplo, el selector es p, que se refiere a todos los párrafos, y el bloque de declaración contiene las propiedades color y font-size.

5. Elementos de una regla CSS

Una regla CSS consta de varios elementos:

  • Regla: El conjunto completo de estilos aplicados a uno o más elementos.
  • Selector: Parte de la regla que identifica los elementos a los que se aplicará el estilo.
  • Bloque de declaración: Contiene las propiedades y valores entre llaves {}.
  • Propiedad: Aspecto del elemento que se desea modificar, como color, font-size, etc.
  • Valor: El valor asignado a una propiedad, como red o 16px.

6. Medios CSS

Los medios CSS, o media queries, permiten aplicar estilos diferentes según las características del dispositivo donde se visualiza la página, como el tamaño de la pantalla. Esto es crucial para el diseño responsivo. Por ejemplo:

@media screen and (max-width: 600px) {
body { background-color: lightblue; }
}

Esto cambiará el fondo a azul claro en pantallas de 600 píxeles de ancho o menos.

7. Comentarios CSS

Los comentarios en CSS permiten añadir notas o anotaciones en el código que no son interpretadas por el navegador. Se utilizan para documentar el código o para desactivar temporalmente estilos. Se escriben entre /* y */. Por ejemplo:

/* Este es un comentario que explica el siguiente estilo */
h2 { color: blue; }

8. Sintaxis de la definición de las propiedades CSS

La sintaxis básica para definir propiedades CSS es la siguiente: se comienza con un selector, seguido de un bloque de declaración que contiene propiedades y valores. La estructura es:

selector {
propiedad: valor;
}

Por ejemplo, para cambiar el color de todos los encabezados h1 a rojo, la regla sería:

h1 { color: red; }
